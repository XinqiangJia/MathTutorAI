# MathTutorAI — 小学数学 AI 教学助手

## 🖥️ 运行环境
- macOS (Apple Silicon, 如 M3/M4)
- Python 3.10+
- Ollama 已安装并运行

## 🚀 快速开始

1. 安装依赖：
   ```bash
   pip install -r requirements.txt


---

## ▶️ 如何在 PyCharm 中运行

1. 打开 PyCharm → **Open** → 选择 `MathTutorAI` 文件夹
2. 右键 `main.py` → **Run 'main'**（命令行交互）
3. 或右键 `gui_app.py` → **Run 'gui_app'**（图形界面）

> 💡 提示：确保终端中已运行 `ollama serve`（通常安装后自动后台运行）

---

## ✅ 优势总结

- **完全离线**：数据不出本地，适合教育隐私场景  
- **Mac 优化**：利用 Apple Silicon 高效推理  
- **双模式**：命令行 + 图形界面，灵活教学  
- **可扩展**：后续可加入题目生成、答题统计等模块
